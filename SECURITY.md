# Security Policy

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, email:

github@trex.ae

Please include:

- a clear description of the issue
- affected versions or commit ranges
- reproduction steps or proof of concept, where possible
- impact assessment
- any suggested remediation

## Response approach

TREX FZE will review security reports and aim to validate, prioritize, and address legitimate vulnerabilities in a reasonable timeframe.

## Disclosure

Please allow TREX FZE reasonable time to investigate and remediate reported issues before public disclosure.
